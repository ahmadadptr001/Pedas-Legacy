  Grid menu interaktif dengan tombol pemesanan langsung ke WhatsApp.

  Kelola pesanan: tambah/kurangi jumlah, hapus produk, dan checkout via WhatsApp.

- **Halaman Kontak**  
  Formulir kontak + info media sosial dan lokasi outlet.
